Het Boxmodel

Bij webdesign is het boxmodel erg belangrijk. Het geeft weer hoe een element weer wordt gegeven op de website door middel van de margin, padding en zijn border. Let op de padding is de toegevoegde ruimte binnen de border van het element, de margin die buiten het element en de border is de rand om het element heen. Als je de width van een element opgeeft, zal de daadwerkelijke breedte van het element die width zijn plus de border, margin en padding aan beide kanten!
Een visualisatie van het boxmodel is in het plaatje nog even weergegeven.

CSS - Een korte samenvatting

Een handige manier om je webpagina op te fleuren is door middel van styling, wat het best gedaan kan worden door CSS stylesheets te gebruiken. Dit zijn aparte .css bestanden die je linkt aan je html pagina door de link tag te gebruiken, zoals al eerder uitgelegd.
In je css bestand geef je informatie weer over hoe bepaalde delen van je webpagina eruit moeten zien. Dit doe je door een selector toe te passen.
Dit zijn selectors op basis van tag (bijv. alle a tags), id met de hashtag, en class met een punt. De classes en id's kun je aan tags toevoegen in je html document als attributes.
Vervolgens zet je tussen accolades eerst het type styling wat je wil toepassen, een dubbele punt, de stylingsinformatie afgesloten met een puntkomma, ook te zien in de afbeelding hiernaast.
De css styling properties zijn nogal vanzelfsprekend, meestal is aan de naam al wel te zien wat het doet, zo bepaald de background-color de achtergrondkleur en de color de tekstkleur.

De Blokkendoos

Bij webdesign heb je soms ook te maken met de zogenaamde blokkendoos, die bestaat uit meerdere blokelementen, de nav, article, footer, header en div elementen. Deze kun je makkelijk positioneren door middel van de float property die je toepast op deze elementen. Door bijvoorbeeld float:left te gebruiken op een element zal dit element naar linksboven drijven totdat het tegen een ander element aanbotst. De volgorde die je gebruikt in je html code voor de verschillende blokelementen is dan altijd van links naar rechts, van boven naar beneden, als je float:left gebruikt.
